## $$ \int k\,dx $$

</HiddenText>

<HiddenText>

## $$ \int x^u\,dx $$

</HiddenText>

<KatexLeft>

```math
\int x^u\,dx = \frac{x^{u+1}}{u+1} + C \; (u \neq -1)
```

</KatexLeft>

<HiddenText>

## $$ \int \frac{1}{x}\,dx $$

</HiddenText>

<KatexLeft>

```math
\int \frac{1}{x}\,dx = ln\,|x| + C
```

</KatexLeft>

<HiddenText>

## $$ \int e^x\,dx $$

</HiddenText>

<KatexLeft>

```math
\int e^x\,dx = e^x + C
```

</KatexLeft>

<HiddenText>

## $$ \int a^x\,dx $$

</HiddenText>

<KatexLeft>

```math
\int a^x\,dx = \frac{a^x}{ln\,a} + C
```

</KatexLeft>

<HiddenText>

## $$ \int sin\,x\,dx $$

</HiddenText>

<KatexLeft>

```math
\int sin\,x\,dx = -cos\,x + C
```

</KatexLeft>

<HiddenText>

## $$ \int cos\,x\,dx $$

</HiddenText>

<KatexLeft>

```math
\int cos\,x\,dx = sin\,x + C
```

</KatexLeft>
